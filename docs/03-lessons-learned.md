# Lessons Learned

## Key Lessons

- Networking and IP consistency are critical.
- Host-Only and NAT must be clearly separated.
- Wazuh gives endpoint truth.
- Security Onion gives network truth.
- Prometheus and Grafana add performance context.
- Nagios prevents confusion about whether the host is alive.
- DC01 transforms the lab into a real enterprise-style environment.
- Documentation is part of the project, not an afterthought.
