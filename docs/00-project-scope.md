# Project Scope

This project builds a complete enterprise SOC lab with Windows, Linux, SIEM, NDR, monitoring, metrics, dashboards, and attack simulation.

## In Scope

- VMware lab
- Host-Only SOC network
- AD DS with DC01
- Win11 domain endpoint
- Wazuh
- Security Onion
- Nagios
- Prometheus
- Grafana
- Kali attack simulations
- incident playbooks
- portfolio website

## Out of Scope

- production deployment
- real public target attacks
- malware execution
- enterprise-scale log volume
