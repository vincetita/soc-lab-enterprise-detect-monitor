# Detection Correlation

## Purpose

Correlation combines evidence from multiple tools into one incident timeline.

## Evidence Layers

| Layer | Tool |
|---|---|
| Attack execution | Kali |
| Endpoint logs | Wazuh |
| Network traffic | Security Onion |
| Metrics | Prometheus |
| Dashboards | Grafana |
| Availability | Nagios |
| Identity | DC01 |
